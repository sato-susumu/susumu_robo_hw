## 製品概要

AmiVoice Front WT01は、株式会社アドバンスト・メディアが提供する**音声認識・音声対話に特化したバッジ型Bluetoothウェアラブルマイク**。2マイクアレイとノイズキャンセリング技術により、騒音環境下でも利用者の声をクリアに集音する。ネックストラップや背面クリップで装着でき、ハンズフリーでの音声入力・対話に適する。

> **引用元**: [AmiVoice Front WT01 製品ページ](https://www.advanced-media.co.jp/products/service/amivoice-front-wt01/) / [MdN 紹介記事](https://www.mdn.co.jp/di/newstopics/55586/)

## マイク・音声仕様

| 項目 | 仕様 |
|------|------|
| マイク方式 | 2マイクアレイ (指向性集音) |
| ノイズ対策 | ノイズキャンセリング / ノイズ除去技術 |
| 耐騒音性能 | 最大 90 dB 環境下で音声認識利用可 |
| 耐騒音性能 (拡張) | 最大 100 dB (専用アダプタ「AmiVoice Front PM01」併用時) |
| スピーカー出力 | 最大 89 dBSPL |
| 抗菌・消毒 | メチルアルコール・次亜塩素酸ナトリウム (1.2%) での消毒に対応 |

> **引用元**: [AmiVoice Front WT01 製品ページ](https://www.advanced-media.co.jp/products/service/amivoice-front-wt01/)

> 90 dBは騒がしい工場やカラオケ店内に相当する騒音レベル。

## 本構成での想定用途

- ロボットへの音声コマンド入力 (騒音環境でも利用者の声をクリアに集音)
- 音声対話インターフェースのマイク入力端末 (Bluetooth経由でRaspberry Pi等に接続)
- ハンズフリーでの操作・指示

> **関連**: 音声入出力の検討としては会議用スピーカーフォン [Anker PowerConf S3](検討中/Anker_PowerConf_S3.md) も参照。WT01はウェアラブル・個人装着型で騒音耐性が高く、用途が異なる。
