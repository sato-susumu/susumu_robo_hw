## LEDインジケーター

> **引用元**: [Anker PowerConf S3 ユーザーマニュアル (ManualsLib)](https://www.manualslib.com/manual/1997951/Anker-Powerconf-S3.html?page=4)

本体には3つのLEDがある。

| LED名 | 配置 | 役割 |
|-------|------|------|
| ステータスLED（部品番号6） | 上面・リング状 | 通話状態・音量・ミュート・バッテリー等を表示 |
| 電源LED（部品番号7） | 側面・電源ボタン付近 | 充電状態・電源ON/OFF状態を表示 |
| Bluetooth LED（部品番号10） | 側面・Bluetoothボタン付近 | Bluetooth接続状態を表示 |

### ステータスLED（上面・リング状）

上面中央のリング形LEDで、時計回り/反時計回りに点灯するなど多様なパターンで状態を表示する。

| 色・パターン | 意味 |
|-------------|------|
| 白色・時計回りに点灯後フェードアウト | 起動完了 |
| 白色・2秒点灯 | USB/電源接続を検出 |
| 緑色・ブリーズ（呼吸するように点滅） | 着信あり |
| 青色・一部点灯 | 音が聞こえた方向を表示 |
| 青色・時計回りに点灯 | 音量UP操作 |
| 青色・反時計回りに点灯 | 音量DOWN操作 |
| 赤色・点灯 | マイクミュート中 |
| 緑色・点灯（点灯数で残量を表示） | 電源ボタン短押し時にバッテリー残量を表示 |

### 電源LED（側面・電源ボタン付近）

電源ボタンに隣接した側面のLED。充電状態と電源状態を示す。

| 色・パターン | 意味 |
|-------------|------|
| 赤色・点灯 | 充電中 |
| 白色・点灯 | 充電完了・電源ON状態 |

### Bluetooth LED（側面・Bluetoothボタン付近）

> **引用元**: [Anker PowerConf S3 ユーザーマニュアル (ManualsLib)](https://www.manualslib.com/manual/3129795/Anker-Powerconf-S3.html?page=5)

Bluetoothボタンに隣接した側面のLED。Bluetooth接続状態を示す。

| 色・パターン | 意味 |
|-------------|------|
| 青色・点滅 | Bluetoothペアリング中 |
| 青色・常時点灯 | Bluetooth接続済み |

---

### バッテリー残量の読み方

> **引用元**: [Anker Japan FAQ](https://www.ankerjapan.com/blogs/faq/powerconf-powerconf-s3-powerconf)

電源がONの状態で電源ボタンを短押しすると、ステータスLEDが緑色に点灯してバッテリー残量を示す。点灯するLEDの数（リングの分割数）で残量を確認できる。具体的な点灯数と残量の対応はマニュアルに明記されていないため、AnkerWorkアプリで数値確認するのが確実。

| 確認方法 | 手順 |
|----------|------|
| 本体LEDで確認 | 電源ON状態で電源ボタンを短押し → ステータスLEDが緑色点灯 |
| アプリで確認 | AnkerWorkアプリ（iOS/Android）を起動 → バッテリー残量が数値で表示 |

## ボタン操作

> **引用元**: [Anker PowerConf S3 ユーザーマニュアル (ManualsLib)](https://www.manualslib.com/manual/1997951/Anker-Powerconf-S3.html?page=4)

| 操作 | 動作 |
|------|------|
| 電源ボタン長押し | 電源ON/OFF |
| 電源ボタン短押し | バッテリー残量確認（LEDが緑色点灯） |
| Bluetoothボタン | Bluetooth ON/OFF切替 |
| Bluetoothボタン2回押し | マルチポイント接続（2台目ペアリング） |
| Bluetoothボタン + 音量－長押し（LED4回点滅まで） | リセット |
| ミュートボタン | マイクミュート ON/OFF |
